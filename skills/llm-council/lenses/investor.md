# Investor lens — capital allocation decisions

Use for investment decisions: whether to invest, position size, when to exit,
portfolio construction. Forces the council to think about downside and
correlation, not just the pitch.

## Sub-questions (answer all four)

### 1. Bull case
- What has to be **true** for this to deliver a 5-10x return?
- What's the **most optimistic but still defensible** outcome — the one that
  doesn't require miracles, just execution plus a tailwind?
- What is the **mechanism** of the upside (revenue growth, multiple expansion,
  strategic acquisition, rerating)? Not all multipliers are equal.

### 2. Bear case
- What's the **most likely** path to losing 50%+ of capital?
- What **permanent impairment** looks like here — not a drawdown, a zero.
- What signal would **pre-warn** the bear case? If there is no pre-warning
  signal, sizing should be smaller than conviction suggests.

### 3. Macro factors
- What **macro regime** does this bet implicitly assume? (Rates, growth, AI
  capex, geopolitical, regulatory?)
- Which regime would **break** the bull case even if execution is perfect?
- Is this a bet the user is making **because** of their macro view, or
  **in spite of** it? Honesty matters here.

### 4. Portfolio fit
- How **correlated** is this with existing holdings? (Factor, sector, regime?)
- Does this **reduce or amplify** portfolio-level risk? (Concentration is
  fine if intentional, bad if accidental.)
- If this goes to zero, is the portfolio still structurally sound, or is
  this position **load-bearing**?
