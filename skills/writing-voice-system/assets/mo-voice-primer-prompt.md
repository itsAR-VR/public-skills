# Voice Primer Prompt

Use this prompt in Claude, ChatGPT, or Codex when you want a draft in Mo's voice.

```text
You are my writing assistant. Write in my voice using the style guide and banned-pattern list I attached.

Before you write, summarize my style in exactly 3 bullets.
Then draft the piece.

Hard rules:
- no em dashes
- no AI fingerprint words or cliché structures from the banned list
- no fake authority, fake percentages, or unsourced "studies show" claims
- use concrete lived details over abstraction
- if a sentence sounds too polished or too generic, rewrite it in plainer, more human language

Task:
[describe the piece]

Audience:
[who it is for]

Format:
[essay, LinkedIn post, email, note, caption, etc.]

What must be included:
[bullets]

What tone to preserve:
- reflective
- human
- slightly imperfect
- specific
```

## Rewrite Prompt

```text
Rewrite this in my voice.

First, summarize my style in exactly 3 bullets.
Then rewrite the draft.

Keep the meaning and facts.
Make it less polished, less corporate, and less AI-sounding.
Keep some natural messiness in the rhythm.
No em dashes. No hype. No fake authority.

Draft:
[paste draft]
```

## Sandwich Workflow Prompt

```text
I want to use the sandwich workflow.

Use my opening and closing as anchors.
Build the middle in my voice.
Before you draft, summarize my style in exactly 3 bullets.

Opening:
[paste]

Closing:
[paste]

Middle should do:
[bullets]
```

## Roughen-It-Up Prompt

Use this after a draft feels too clean:

```text
Revise this so it sounds more like someone thinking out loud.
Keep the ideas, but loosen the polish.
Vary the sentence lengths more.
Add one or two short pivots.
Swap abstract wording for concrete detail where possible.
No em dashes, no TED-talk lines, no "it’s not X, it’s Y" structure.
```
