# Hormozi Skill: Retention

Use when: churn is high, LTV is flat, or customer engagement is dropping.

## Core principles
- Retention is a system: proactive monitoring beats reactive save attempts.
- Engagement decline is an early churn signal; intervene before cancellation.
- Simple human touches (recognition/events/cards) compound retention and referrals.
- Exit conversations can recover meaningful churn when process is enforced.
- Billing and pricing model choices directly impact churn and LTV.

## Key frameworks
### 1) 5 Horsemen of Retention (from playbook examples)
1. **Track attendance/usage** (watch decline patterns)
2. **Reach out 2x/week** (praise progress + solve small issues)
3. **Handwritten cards** (onboarding, milestones, holidays)
4. **Member events cadence** (community touchpoints)
5. **Exit interviews before cancellation finalization**

### 2) Churn Checklist workflow
- Define leading indicators (attendance/engagement/product usage).
- Trigger outreach at threshold drop.
- Run save intervention.
- If still cancelling, run structured exit interview.
- Capture reason codes and feed product/onboarding improvements.

### 3) LTV-aware retention loop
- Improve onboarding speed-to-value.
- Segment customers by engagement level.
- Deliver interventions by segment.
- Use retention wins to generate referrals and resells.

## Practical templates
### Template A: Attendance trigger SOP
- Trigger: `[usage drops from X to Y]`
- Owner: `[CS rep]`
- Action in 24h: `[check-in call/text + small win assignment]`
- Success criteria: `[usage back to Z within N days]`

### Template B: Exit interview opener
`Before we finalize cancellation, I want to understand what changed so we can either fix it now or improve in the future. What’s the main reason?`

### Template C: Milestone engagement cadence
- Day 1: welcome + first-win action
- Day 7: progress check
- Month 1/3/6/12: milestone message + referral ask

## Examples
1. Gym/member business:
- Usage dropped from 3x/week to 2x/week → immediate outreach prevented churn.

2. SaaS/onboarding:
- Added first-value checklist in week 1 → reduced early-month churn.

3. Service company:
- Exit interview became mandatory before final cancellation; save rate improved.
