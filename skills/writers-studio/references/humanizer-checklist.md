# Humanizer Checklist

Condensed from the full humanizer skill (based on Wikipedia's "Signs of AI writing" guide).
This is the mandatory second pass for all writers-studio output.

## Scan order

Run these checks in sequence. Fix every violation before outputting.

### Language kills
| Pattern | Example | Fix |
|---------|---------|-----|
| Inflated significance | "serves as a testament" | State the fact plainly |
| Superficial -ing phrases | "highlighting the importance" | Direct statement or cut |
| Promotional language | "boasts a vibrant ecosystem" | Specific concrete detail |
| Vague attributions | "Industry experts believe" | Name the source or cut |
| AI vocabulary | delve, foster, tapestry, landscape | Simpler word |
| Copula avoidance | "serves as" instead of "is" | Use "is/are/has" |
| Filler phrases | "In order to," "It is important to note" | Simplify or cut |
| Excessive hedging | "could potentially possibly" | Commit or cut |

### Structure kills
| Pattern | Example | Fix |
|---------|---------|-----|
| Negative parallelism | "Not only X, but Y" | Just state Y |
| Rule of three | "innovation, inspiration, and insights" | Two items, or restructure |
| Elegant variation | protagonist/main character/central figure/hero | One word, reuse it |
| False ranges | "from X to Y" on unrelated scale | List directly |
| Inline-header lists | **Bold:** followed by explanation | Flowing prose or plain bullets |

### Style kills
| Pattern | Example | Fix |
|---------|---------|-----|
| Em dashes | Multiple — per piece | Commas, periods, colons. AR uses zero em dashes. |
| Bold overuse | **every** **other** **word** | Reserve for true emphasis only |
| Title case headings | "Strategic Negotiations And Partnerships" | Sentence case |
| Emoji decoration | 🚀 Launch Phase | Remove all emoji |
| Curly quotes | "like this" | Straight quotes "like this" |

### Tone kills
| Pattern | Example | Fix |
|---------|---------|-----|
| Sycophancy | "Great question!" | Cut entirely |
| Collaborative artifacts | "I hope this helps!" | Cut unless natural |
| Generic conclusions | "The future looks bright" | Specific detail or just stop |
| Knowledge disclaimers | "As of my last update" | Cut entirely |

## After fixing

Read the output aloud (mentally). Does it:
- Sound like a person wrote it?
- Vary sentence length naturally?
- Have opinions, not just neutral reporting?
- Use specific details over abstract claims?

If yes, it passes. If not, rewrite again.
